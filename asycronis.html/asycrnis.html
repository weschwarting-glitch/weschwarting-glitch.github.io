Accessibility

The DOM order keeps the content first and sidebar second, which keeps reading order logical. I did not use the order property so keyboard focus stays correct. The layout stacks naturally on small screens, making it easier to read on mobile.

exit ticket

The most reusable pattern was the wrapping card layout because it works for many designs. Flexbox made alignment and spacing easier without using hacks. Using a gap helped keep spacing clean. Flexbox is great for one-direction layouts like rows or columns. Grid would be better for full page layouts with rows and columns at the same time. Next time I would test more screen sizes to improve responsiveness. I would also refine spacing and sizing for better balance.
